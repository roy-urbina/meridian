☁️ Sync Data — GitHub Gist

Backup and sync your Meridian data across devices
How this works: Your data is saved as a private file on your GitHub account (called a Gist). Only you can see it. To sync between devices, open Meridian on each device and paste the same token and Gist ID.

One-time setup: Create a free GitHub account, then generate a personal access token with Gist permissions. Takes about 2 minutes. See the instructions below.
This is stored only in your browser's localStorage and never sent anywhere except GitHub's own API.
📖 Step-by-step setup instructions ▶
Step 1 — Create a GitHub account
Go to github.com and sign up for a free account if you don't have one. Confirm your email.

Step 2 — Generate a Personal Access Token
1. Click your profile picture (top right) → Settings
2. Scroll all the way down → click Developer settings
3. Click Personal access tokens → Tokens (classic)
4. Click Generate new token → Generate new token (classic)
5. Give it a name like "Meridian Sync"
6. Set expiration to "No expiration" (or however long you want)
7. Check the gist checkbox — that's the only permission needed
8. Click Generate token at the bottom
9. Copy the token immediately — GitHub only shows it once. It starts with "ghp_"

Step 3 — Paste the token above and click Push to Cloud
Leave the Gist ID blank on your first sync. Meridian will create a new private Gist automatically and show you the ID to save.

Step 4 — Set up on your other devices
Open Meridian on your phone or another computer. Paste the same token and the Gist ID that was created in Step 3. Click Pull from Cloud to load your data.
